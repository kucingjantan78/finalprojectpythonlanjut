Nama Program: Antivirus No Debat

Deskripsi Singkat:
Program Antivirus Python ini dibuat sebagai bagian dari tugas Ujian Akhir Semester (UAS) pada mata kuliah 
Pemrograman Python Lanjut. Program ini menggunakan antarmuka grafis menggunakan modul Tkinter untuk memindai 
file dalam suatu direktori dan memeriksa apakah file tersebut aman atau tidak dengan menggunakan layanan 
VirusTotal API. Hasil pemindaian ditampilkan dalam antarmuka grafis pengguna, mencakup informasi hash SHA256, 
tautan hasil pemindaian, dan status keamanan file.

*Fitur Utama:*
1. *Pemindaian File:* Program dapat memindai file dalam direktori yang dipilih oleh pengguna.
2. *Penggunaan VirusTotal API:* Program menggunakan VirusTotal API untuk memeriksa keamanan file.
3. *Tampilan Hasil Pemindaian:* Hasil pemindaian, termasuk informasi hash dan tautan hasil, ditampilkan dalam antarmuka pengguna.
4. *Informasi Keamanan:* Program memberikan informasi tentang apakah file dianggap aman atau tidak, berdasarkan hasil pemindaian dari antivirus terkait.

*Cara Menggunakan:*
1. Buka program.
2. Pilih direktori yang ingin Anda periksa dengan mengklik tombol "Browse Folder".
3. Klik tombol "Scan Files" untuk memulai pemindaian file dalam direktori.
4. Lihat hasil pemindaian yang ditampilkan di antarmuka pengguna.

*Persyaratan:*
- Python 3.10.X
- Modul Tkinter
- Modul Requests

*Penting:*
Pastikan untuk mengganti nilai api_key dengan kunci API yang valid dari VirusTotal sebelum menjalankan program.
